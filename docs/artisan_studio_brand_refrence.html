Type hierarchy (simple)

  • H1: Sans 700 (or 600 for softer)
  • H2/H3: Sans 600
  • Body: Sans 400
  • Buttons: Sans 600
  • Quotes: Fraunces 400–600
  • Data: Plex Mono 400–500

Rules of thumb

  • Prefer weight changes over switching fonts.
  • Keep mono for “data-ish” things, not general copy.
  • If a page feels too serious, add a short Fraunces line — then stop.

Where this shows up

  • Nav + UI: Source Sans 3
  • Testimonials: Fraunces
  • Pricing/spec tables: IBM Plex Mono
